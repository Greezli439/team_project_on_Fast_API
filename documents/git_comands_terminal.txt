GitTutorial Terminal

1.	git clone – копіює репозиторій (папку з файлами) на комп’ютер
2.	git branch “Name” - створює гілку з назвою “Name”
3.	git checkout “Name” – переходить на гілку з назвою “Name”
4.	git checkout – b “Name” – створює гілку з назвою “Name” та переходить на неї
5.	git add . – зберігаємо зміни в файлах
6.	git commit -m “commit massage” – підписуємо збережені зміни в файлах
7.	git push – відправляємо зміни на сайт github
8.	git pull – отримаємо останні зміни з сайту github
9.	git status – показуємо статус проекту
10.	git branch – показуємо список гілок проекту
11.	git branch -r – показує список гілок на сайті github
12.	git branch -a – показує список гілок на комп’ютері та на сайті github
13.	git fetch – отримаємо зміни з сайту github
14.	git stash – зберігаємо не збережені зміни в файлах і кладемо їх в буфер обміну
15.	git stash apply – встановлюємо збережені зміни з буферу обміну
16.	git merge “Name” – зливаємо гілку з назвою “Name” в поточну гілку
17.	git merge –about – відміняємо зливання гілок
18.	git branch -d “Name” – видаляє гілку локально з проекту (з комп’ютера)
19.	git push origin –delete “Name” – видаляє гілку з сайту github
20.	git diff – показує відрізки рядків між двома версіями файлу (між двома комітами)
21.	git log – показує історію комітів

Для того щоб вийти з режиму перегляду комітів використовуйте клавішу q
